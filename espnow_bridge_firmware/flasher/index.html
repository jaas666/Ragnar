RAGNAR — ESP-Now Coordinator

Flash Coordinator

Standalone Piglet Core · ESP-Now mesh · HuginnESP-compatible JSON output

Two supported boards. Both pair automatically with Piglet nodes on channel 6 and stream WiGLE/BLE records to Ragnar over USB serial as one-object-per-line JSON.

ESP32-C5 Coordinator (headless)

Waveshare ESP32-C5-WIFI6-KIT · single-core RISC-V @240 MHz, 16 MB flash, 4 MB PSRAM, dual-band Wi-Fi 6, BLE 5. No display. Smallest/cheapest option — just plug into Ragnar's USB and it works.

Before flashing
  1. Close Arduino IDE, Serial Monitor, or any other app using the serial port
  2. Use a data-capable USB-C cable (not charge-only)
  3. Click Flash and select USB JTAG/serial debug unit from the port list
  4. If the board isn’t detected, hold BOOT while pressing RST
Waveshare ESP32-C5-WIFI6-KIT

ESP32-S3-Touch-LCD-4B Coordinator (480×480 display)

Waveshare ESP32-S3-Touch-LCD-4B · dual-core LX7 @240 MHz, 16 MB flash, 8 MB OPI PSRAM, 4″ 480×480 RGB IPS touch panel. Same coordinator firmware, plus a status UI on the built-in display (nodes, networks seen, uptime, last RX).

Before flashing
  1. Close Arduino IDE, Serial Monitor, or any other app using the serial port
  2. Use a data-capable USB-C cable (not charge-only)
  3. Click Flash and select USB JTAG/serial debug unit from the port list
  4. If the board isn’t detected, hold BOOT while pressing RST
Waveshare ESP32-S3-Touch-LCD-4B

📡 Serial Monitor

Open a browser-based serial monitor for any flashed coordinator. Default baud is 460800. Expect [CORE] alive lines every 10 s plus {"type":"WIFI",...} JSON rows when paired Piglets are reporting.

Web Serial is only available in Chromium-based browsers (Chrome, Edge, Opera) on desktop. Firefox and Safari are not supported.
Channel: 6 · Baud: 460800 · Output: HuginnESP-style JSON
