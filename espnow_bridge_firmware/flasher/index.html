Forge the Coordinator

Piglet Core  ⚔  ESP-Now mesh  ⚔  HuginnESP-compatible JSON

Two longships to choose from. Either board pairs automatically with the Piglet mesh on channel 6 and herds WiGLE / BLE records back to Ragnar over USB serial as one-object-per-line JSON. Pick a board, plug it in, raise the sails.

🐖 ESP32-C5 Coordinator (headless · the scout)

Waveshare ESP32-C5-WIFI6-KIT · single-core RISC-V @240 MHz, 16 MB flash, 4 MB PSRAM, dual-band Wi-Fi 6, BLE 5. No display, no fuss — the smallest, lightest pig in the sty. Plug it into Ragnar's USB and it starts oinking immediately.

Before the forge fires
  1. Close Arduino IDE, Serial Monitor, or any other app holding the port
  2. Use a data-capable USB-C cable (charge-only cables are runes that do nothing)
  3. Click Forge and pick USB JTAG/serial debug unit from the port list
  4. If the board isn’t spotted, hold BOOT while tapping RST
Waveshare ESP32-C5-WIFI6-KIT

🐗 ESP32-S3-Touch-LCD-4B Coordinator (480×480 display · the jarl)

Waveshare ESP32-S3-Touch-LCD-4B · dual-core LX7 @240 MHz, 16 MB flash, 8 MB OPI PSRAM, 4″ 480×480 RGB IPS touch panel. Same coordinator firmware, plus a glowing rune-board on the built-in display (mesh nodes, networks heard, uptime, last RX).

Before the forge fires
  1. Close Arduino IDE, Serial Monitor, or any other app holding the port
  2. Use a data-capable USB-C cable (charge-only cables are runes that do nothing)
  3. Click Forge and pick USB JTAG/serial debug unit from the port list
  4. If the board isn’t spotted, hold BOOT while tapping RST
Waveshare ESP32-S3-Touch-LCD-4B

📡 Skald’s Ear (serial monitor)

Open a browser-based serial monitor for any forged coordinator. Default baud is 460800. Once awake, the board announces itself with {"device":"RagnarCoord","fw":"c5-1","board":"ESP32-C5",...}, then emits [CORE] alive every 10 s and one {"type":"WIFI",...} / {"type":"NODE",...} line per record as paired Piglets report in.

Web Serial only sails in Chromium-based browsers (Chrome, Edge, Opera) on desktop. Firefox and Safari are unworthy ships for this voyage.
Channel: 6 · Baud: 460800 · Output: HuginnESP-style JSON
